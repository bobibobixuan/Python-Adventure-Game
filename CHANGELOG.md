# 更新日志

## 2026-05-14

### 1.0.4
- 成就弹窗从屏幕居中移至右下角，不再遮挡答题题目，沿用现有淡入淡出动画体系。
- 成就弹窗支持多弹窗堆叠，最多同时显示 3 个，超出时自动移除最旧的，各弹窗垂直排列互不重叠。
- 新增 WebSocket 实时在线状态系统：学生登录后自动建立 WebSocket 每 30 秒心跳，关闭页面自动离线。
- 教师后台仪表盘新增「当前在线」卡片，实时显示在线人数/总人数，点击可展开在线/离线学生名单。
- 教师后台学生列表新增「状态」列，每行学生昵称前显示绿色（在线）或灰色（离线）圆点。
- WebSocket 连接支持指数退避自动重连，网络中断恢复后无需手动刷新。
- 后端新增 `/ws/online` 端点，Token 通过首条消息传递（非 URL 参数），避免密钥泄露到服务器日志。

### 1.0.3
- 修复游戏前端登录后仍主要使用本地 `gameState` 与 `localStorage`、没有把学习进度同步到数据库的问题。
- 新增 `/api/records/sync-state`，登录后与本地保存时会把总答题数、总分、连击、极限测试次数、单元关卡解锁和星级同步到后端。
- 调整游戏前端答题流程：已登录用户每次答题都会实时提交 `AnswerRecord` 到后端，教师后台与数据库统计不再长期脱节。
- 调整教师后台仪表盘总答题数统计，优先兼容已同步到 `UserStats` 的累计答题数据。

## 2026-05-13

### 1.0.2
- 修复教师后台左侧菜单在自动恢复管理员登录态后无法切换页面的问题。
- 调整教师后台初始化顺序，先绑定侧边栏导航事件，再恢复登录态，避免 `showContent()` 提前返回导致菜单失效。
- 为教师后台静态资源追加版本参数，避免浏览器继续复用旧版 `admin.js` / `admin.css` 缓存。

## 2026-05-12

### 1.0.1
- 修复 `dev/index.html` 与 `build.py` 打包协议不一致的问题，恢复 `BUILD` 占位符，确保发布版 HTML 会内联样式和脚本。
- 重新生成发布文件，修复打开 `dist` 目录中的正式版页面时样式缺失的问题。

## 2026-04-21

### 新增
- 新增开发者口令验证弹层，统一开发者模式入口。
- 新增开发者控制台总览卡片，可快速查看积分、做题数、星级、成就和极限测试统计。
- 新增按单元切换的开发者操作区，支持单元解锁、满星和重置。
- 新增全局进度操作，支持全部单元一键解锁、全部满星和整体重置。
- 新增成就与错题管理功能，支持按单元清理错题和全量清理。
- 新增统计数据重置与默认存档恢复功能。

### 优化

- 移除对固定 6 关结构的依赖，改为按单元实际关卡数动态处理。
- 优化开发者操作反馈，状态区现在会显示时间戳。
- 优化开发者界面样式与响应式布局，兼容桌面端和移动端。
